
# VScode
## 初始配置
> 配置 git 账号和密码
- 首选项-->设置，搜索 git.path，输入 'E:/Program Files/Git/bin/git.exe'
- git config --global credential.helper store

Shift + Alt + U 将配置环境 扩展插件信息上传到云端

Shift + Alt + D 下载云端的配置

## 快捷键

> 多光标同时编辑

-  按住 alt 键 单击，可选中多个位置
- cmd + shift + L 选中多个相同的内容
