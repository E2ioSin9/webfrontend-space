封装一个replace
```javascript
String.prototype.myReplace = function myReplace(reg, callback) {
    //=>this:str
    //=>默认REG肯定加G了,CALLBACK肯定传递函数了
    //可以用!reg.global来写不加g的情况
    let _this = this,
        val = reg.exec(this);
    while (val) {
        //=>res:每一次EXEC捕获的结果（数组）
        let res = callback(...val),//'陈小陈哈哈'  '28'
            i = _this.indexOf(val[0]);//为什么不用val.index？
        _this = _this.substring(0, i) + res + _this.substring(i + val[0].length);
        val = reg.exec(this);
    }
    return _this;
};
let str = 'my name is {0}，i am {1} years old!',
    ary = ['陈小陈哈哈', '28'];
str = str.myReplace(/\{(\d+)\}/g, function (...arg) {
    let index = arg[1];
    return ary[index];
});
console.log(str);
```

# 常见面试题
## 判断字符串以字母开头，后面字母数字下划线，长度6-30
```js
const reg = /^[a-zA-Z]\w{5, 29}$/
```
常用的简单的正则表达式：
```js
邮政编码：/\d{6}/
小写英文字母：/^[a-z]+$/
英文字母：/^[a-zA-Z]+$/
日期格式：/^\d{4}-\d{1, 2}-\d{1, 2}$/
简单的IP地址匹配：/\d+\.\d+\.\d+\.\d+/  // .匹配任意字符
```
## 手写字符串 trim 方法
```js
String.prototype.trim = function () {
   return this.replace(/^\s+/, '').replace(/\s+$/, '')
}
```
